V14 - Registration / Forgot Password / Phone Login

Backend:
- Khi khách đăng ký tài khoản, hệ thống gửi mail về SUPPORT_EMAIL, mặc định support@posora.vn.
- Đăng ký có thể dùng số điện thoại làm username nếu khách không nhập tên đăng nhập.
- Email không bắt buộc.
- Quên mật khẩu: gửi mã qua email hoặc SMS nếu cấu hình provider.
- Đăng nhập OTP bằng số điện thoại.

ENV cần cấu hình để gửi mail:
SUPPORT_EMAIL=support@posora.vn
SMTP_HOST=...
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=...
SMTP_PASS=...
SMTP_FROM=MeatBiz <support@posora.vn>

Nếu muốn gửi SMS:
SMS_PROVIDER_URL=https://...
SMS_PROVIDER_API_KEY=...

SQL cần chạy:
backend/sql/V14_REGISTRATION_FORGOT_PASSWORD_MAIL_PHONE.sql

Tiếp theo sau phần này có thể áp dụng OpenClaw vào MeatBiz theo hướng:
- OpenClaw Agent Orchestrator
- Tool registry cho Customer/Product/Order/Inventory/Debt/Supplier tools
- Agent memory + audit action timeline
