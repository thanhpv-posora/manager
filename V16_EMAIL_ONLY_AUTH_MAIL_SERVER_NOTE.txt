V16 Email-only Auth Mail Server

- Registration requires email verification.
- New registration sends verify email to customer.
- New registration sends notification email to support@posora.vn.
- Forgot password sends reset code by email.
- Phone-only registration / phone OTP shows "đang phát triển".
- SMS OTP is disabled because no free stable SMS provider is configured.
