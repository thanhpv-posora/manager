# Claude Playbook: One Task, One Session

Due to context and compaction limits on Windows, use Claude Code with one task per session.

## Workflow

1. Start Claude Code.
2. Give one small task.
3. Get report.
4. Exit.
5. Open new session for next task.

## Do Not

- Do not keep one Claude session for hours.
- Do not rely on `/compact`.
- Do not ask Claude to review the entire repository repeatedly.
- Do not allow edits unless task explicitly requires implementation.
